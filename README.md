**INTEGRAÇÃO REST API USANDO PHP**

Esta integração foi desenvolvida para enviar o cadastro de alunos que realizasse a inscrição para o vestibular, 
desta forma, a plataforma vestibular online receberia os dados e aplicaria a prova para o aluno.

Esta integração foi criada para Faculdade Metropolitana de Anápolis - FAMA, no segundo semestre de 2020.
